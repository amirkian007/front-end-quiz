# Form Submition

## Description

in `App.vue` we have a simple form with three inputs made with vue 3x.The goal is to display the form information in an alert upon form submission. 
Handle the form data and submission by <ins>**leveraging browser features**</ins> and display the information in an alert.

## Getting started
```bash
#install pnpm globally if not installed
npm i pnpm -g
#install packages
pnpm i 
#run the dev server on local host
pnpm run dev
```
